Pokedex

Descrição

Um site criado com objetivo de estudar APIs, requisições e responsividade. A ideia é usar a uma API pública para criar um site onde mostra as informações dos pokémons(nome, imagem e número do pokemon). Sendo possivel realizar uma pesquisa usando seu nome, mostrando o resultado da pesquisa na tela. Para limpar a pesquisa, basta clicar no botão “pesquisar” com a barra de pesquisa em branco.

Foi um mini projeto satisfatório, sempre há como melhorar, tanto o Javascript quanto o design da página, mas como dito anteriormente, apenas treinar o uso de APIs, requisições e responsividade.
